# MCQ Paper Generator

Create, manage, and export MCQ papers with automatic answer keys.

## Run from Source

```bash
python app.py
```

## Ready-to-Use Executable (macOS)

Location: `dist/`

- `MCQ_Paper_Generator` (main executable)
- `MCQ_Paper_Generator.app` (app bundle)
- `license.json` (keep in the same folder)

Run:

```bash
cd dist
./MCQ_Paper_Generator
```

If macOS blocks the app: right-click → Open → Open. If needed: `xattr -cr MCQ_Paper_Generator`.

## License

The application requires `license.json` in the executable folder. A license is already embedded, but the file is also provided in `dist/` for convenience.
